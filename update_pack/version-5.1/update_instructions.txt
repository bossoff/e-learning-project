 - Before updating to version 5.1, make sure that you currently running your product on version 5.0.

Let's proceed,

- First, upload the razorpay-php.zip library upload in your server manually and extructed. You will find it inside the version-5.1 directory.
- Uploading location: Root directory in your server > application > libraries > razorpay-php

- Now, go to system settings from settings menu in admin panel
- Move to the section 'Update Product' on that page
- Upload the file 'update_5.1.zip' which is located at update_pack > version-5.1 folder inside your copy of Academy app downloaded from Codecanyon.
- Hit the button 'Update'.
- That's it
